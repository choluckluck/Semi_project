# 👠 Sue Shop 👠
- 프로젝트 선정 이유 : 사용자가 쉽게 접할 수 있는 쇼핑몰의 다양한 기능을 구현해 보기 위함
- 프로젝트 목표
1) 사용자가 쉽게 접할 기회가 많은 사이트를 구현
2) 기본적인 기능을 구현하고 예외 케이스를 고려하며 개발


<Br>
<h3> 프로젝트 소개 </h3>
- HTML, CSS, JAVASCRIPT, JAVA, OARCLE 등 프로그래밍 언어들을 활용하여 JSP의 MVC 패턴 기법을 적용한 쇼핑몰 웹 사이트
<br>
